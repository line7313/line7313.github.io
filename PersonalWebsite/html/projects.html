Projects

Project 1: FPGA Signal Filter

For ENGR 431 Design of Embedded Systems I used a hardware description langauge (Verilog) on a Zedboard to create a signal filter. The end result was a smooth signal from a noisy input.

Project 2: Home Security System

In ENGR 432 Real Time Systems I helped build a home security system. This system had a display and monitored for movement and detected fires. Once triggered, entering the correct pin via a keypad would disable the alarm.

Project 3: Microprocessor LCD

For ENGR 432 I displayed messages on a 7-segment LCD using an ARM Cortex-M4 microprocessor. Instructions were written in C and bit masking was used to manipulate control register values. This project required careful reading of the processor reference manual.
