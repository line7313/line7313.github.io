Project Experience


Bank Security System - Designed and implemented a hardware security system that utilized three sensors corresponding to different security levels. The highest tripped security level was displayed to the user via a 7-segment display. A reset input was created to initialize the system.


Stack Based Interpreter - Wrote a stack based interpreter program in Java. User input that is not an operation is pushed onto the stack as data. Recognized commands operate on the data within the stack. An extended version of the interpreter was also created and prompts the user to choose which they would like to run from the driver file. This version adds onto the base functionality by using inheritance and adds four basic math operations.


Microprocessor LCD - Displayed messages on a 7-segment LCD using an ARM Cortex-M4 microprocessor. Instructions were written in C and bit masking was used to manipulate control register values. This project required careful reading of the processor reference manual.


FPGA Clock - Used a Basys 3 field programmable gate array to create a clock that displayed the current time on a 7-segment display and PM/AM using an LED. Instructions were written in Verilog. A clock divider was used to create a 1 Hz clock that controlled the least significant digit of the display. Finite state machines were used to handle rollovers of digits. A relatively slow secondary clock was created to handle the enable for each segment digit place. Despite only one display segment being enabled at a time, the secondary clock frequency driving this switching was adjusted to take advantage of the persistence of vision while avoiding glitching.
